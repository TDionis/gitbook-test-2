# BBox to Polygon

This layer \(`bbox2poly`\) converts rectangles \("bounding boxes"\) to polygons.

```javascript
{
  "action": "bbox2poly",
  "src": ["$sample1"],
  "dst": "$sample2",
  "settings": {
    "classes_mapping": {
      "awesome-rect": "awesome-poly"
    }
  }
}
```
