This layer (`merge_bitmap_masks`) takes all bitmap annotations which has same class name and merge it into single bitmap annotation.

```json
{
  "src": [
    "$sample1"
  ],
  "dst": "$sample2",
  "action": "merge_bitmap_masks",
  "settings": {
    "class": "Bird"
  }
}
```
